# OIBSIP

Collection of Java desktop applications built using Core Java and Java Swing.

## Projects

### 1. Smart ATM Management System
A Java Swing based ATM simulator with authentication, deposits, withdrawals, transfers, transaction history, receipt generation, and PIN management.

### 2. Smart Number Guessing Game
A Java Swing based number guessing game featuring adaptive difficulty, achievements, statistics tracking, hints, scoring, and multiple difficulty levels.

## Technologies Used

- Java 17
- Java Swing
- Object-Oriented Programming
- Collections Framework
- Exception Handling
- Git & GitHub
